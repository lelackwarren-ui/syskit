# syskit
# Dans ce repo git, nous allons creer un projet Bash pour ameliorer le deploiment de l'environnement, suivi de l'espace disque, monitorign des processus, navigation rapide dans un terminale Linux.
# Dans le repo syskit nous aurons un script shell pour les fonctions( nommée function.sh ), un script pour les alias( nommée aliases.sh ) et un autre (nommée install.sh)
# Le script functions.sh est pour des fonctions Bash réutilisable. 
# Le script aliases.sh est pour les alias de commandes.
# Le script install.sh est un script d'installation automatiques.
